🤖 Personal Assistant Agent — Intake

Help us build an AI agent tailored to you. Answer honestly — there are no wrong answers.

1. About You

Basic info so the agent knows who it's working with.

Job title, business, freelancer, etc.

2. How You Work

So the agent matches your style, not fights it.

Direct and brief? Detailed? Casual? Formal?

Google Workspace, Notion, GHL, Slack, etc.

Hours, days, when you're most productive

3. Goals & Priorities

What should the agent help you accomplish?

Bottlenecks, time sinks, recurring frustrations

4. Your Strengths & Assets

What do you bring to the table? This helps the agent amplify your strengths.

Skills, knowledge areas, natural talents — be honest

Industry contacts, communities, partnerships, etc.

Content, courses, email lists, IP, real estate, capital, audience, etc.

The thing that gives you an edge others don't have

5. Personality & Boundaries

So the agent feels right, not robotic.

Casual friend? Strict executive assistant? Strategic advisor? Hype man?

Over-explaining? Being too casual? Repetition?

Topics to avoid, hours it shouldn't message, etc.

6. Anything Else

Anything we didn't ask that you want the agent to know.

Your responses go directly to Sami. Nothing is stored publicly.
